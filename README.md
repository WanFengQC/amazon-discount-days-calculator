# 亚马逊折扣天数计算器

用于亚马逊折扣排期模拟、90天合规校验、飞书提醒的本地/局域网网页工具。

## 功能

- 导入 Excel 促销记录（支持 `促销N` + 右侧结束日期列）
- 按产品（ASIN | SKU）切换查看
- 未来日历模拟（可点选折扣日）
- 90 天约束校验：任意 90 天内折扣日 `< 45`（最多 44 天）
- 自动推荐模式：
  - 均匀分布（按 7 天连续块）
  - 中间优先均匀分布
  - 中间优先
  - 头部优先
  - 尾部优先
  - 自定义区间均匀分布
- 历史面板（可折叠，固定最近 90 天，样式与未来日历一致）
  - 折扣日显示为“已选”
  - 支持编辑（点击切换）
- 跨天归档
  - 未来模拟中已过去的折扣日自动归档到历史记录
- 飞书提醒
  - 连续折扣区间只提醒首日
  - 提前 2 天提醒
  - 若 D-2 是周末，提前到周五
  - 支持卡片消息（失败自动回退文本）

## 技术栈

- 前端：原生 HTML/CSS/JavaScript
- 后端：Python + FastAPI
- Excel：SheetJS（浏览器端）
- 持久化：本地 JSON 与上传文件

## 运行

### Windows

```powershell
chcp 65001
.\start.ps1
```

或

```bat
start.bat
```

默认端口：`8010`

- 本机：`http://127.0.0.1:8010/`
- 局域网：`http://<LAN-IP>:8010/`

### Docker

```bash
docker compose up --build
```

## 关键文件

- `index.html`：前端页面与交互
- `backend.py`：后端 API、计算、飞书提醒
- `shared_state.json`：共享业务状态
- `data/feishu_config.json`：飞书配置
- `data/source_import.xlsx`：导入源文件副本

## 飞书说明

- 使用 `app_id + app_secret` 获取 `tenant_access_token`
- 默认从应用可见范围拉取成员（scopes）
- 成员选择支持多选下拉
- 发送提醒前会自动保存当前飞书配置

## 常见问题

- `feishu disabled`：请将“启用提醒”切换为启用并保存（当前版本发送按钮会自动保存一次）
- 局域网无法访问：确认后端绑定 `0.0.0.0` 且防火墙放行 8010
- 成员无头像/昵称：通常是飞书应用权限范围限制，需补齐通讯录可读权限
